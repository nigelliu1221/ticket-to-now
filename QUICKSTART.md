## 3️⃣ 使用流程

### 步驟 1：開啟網頁
```
雙擊 index.html
  ↓
瀏覽器自動開啟
  ↓
看到首頁和隨機文案
```

### 步驟 2：輸入票號
```
輸入 1-500 之間的數字
  ↓
按 Enter 或點擊「開始」
  ↓
看到票卡內容
```

### 步驟 3：呼吸練習（可選）
```
點擊「💨 呼吸練習 30 秒」
  ↓
跟隨視覺引導
  ↓
放鬆、專注、回到當下
```

### 步驟 4：繼續探索
```
點擊左上角 ← 返回
  ↓
輸入下一個票號
  ↓
繼續探索
```

## 5️⃣ 呼吸練習說明

### 自動選擇
- 每天自動隨機選擇一種呼吸法
- 確保練習的多樣性

### 視覺引導
- 🟣 圓球放大 = 吸氣
- 🟣 圓球縮小 = 吐氣
- 🟣 圓球靜止 = 停頓（部分呼吸法）

### 時長
- 固定 30 秒
- 自動倒數計時
- 完成後自動結束

### 10 種呼吸法
1. 平衡呼吸（4-0-4）- 平衡身心
2. 放鬆呼吸（3-0-6）- 深度放鬆
3. 活力呼吸（2-0-2）- 提升活力
4. 深度呼吸（5-0-5）- 冥想練習
5. 專注呼吸（3-2-3）- 提升專注
6. 快速呼吸（2-0-3）- 快速調整
7. 方形呼吸（4-2-4）- 穩定情緒
8. 緩慢呼吸（4-0-6）- 舒緩壓力
9. 延長呼吸（3-0-5）- 減輕焦慮
10. 節奏呼吸（3-1-4）- 建立節奏
